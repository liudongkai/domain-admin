# 获取域名列表

1、请求地址：/api/getDomainList

2、请求方式：POST

3、请求参数

| 参数  | 类型   | 必须 | 说明 |
| -| - | - | - |
|page | int | 否 | 页码，默认1
|size | int | 否 | 每页数量，默认10
|group_id | int | 否 | 分组id

4、返回参数

参见：[getDomainById.md](/domain/getDomainById.md)

5、请求示例

```
POST {{baseUrl}}/api/getDomainList
Content-Type: application/json
X-Token: <token>

{
  "page": 1,
  "size": 1
}
```
